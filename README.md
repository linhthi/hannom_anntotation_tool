### Backend
Install packages:
```
pip install -r requirements.txt
```
Run backend:
```
python app.py
```

### Frontend
Move to frontend directory:
```
cd front_end/tool_annotator
```
Install all dependency:
```
npm install
```
Start front_end:
```
npm start
```
